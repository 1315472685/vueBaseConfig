Proxy 跨域代理

配置入口：config/index.js 

proxyTable: {
  '/api': {
    target: 'http://139.199.201.51:8089', // 接口地址
    changeOrigin: true,
    pathRewrite: {
      '^/api': '/' // 重命名 api 等于上面target的地址
    }
  }
}

反例：

url: 'http://139.199.201.51:8089/user/login' // 默认调用接口跨域

正例：

url: '/api/user/login' // /api/ 调用不跨域

如项目中出现多个接口地址的情况，可联系后台通过 Nginx 处理

Tips：修改反向代理地址后需要重启程序服务才会生效
