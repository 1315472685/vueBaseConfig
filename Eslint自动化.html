ESlint 搭配 vsCode 自动格式化校验方案

安装编程工具 Visual StudioCode (简称 vsCode)

官网地址：https://code.visualstudio.com

安装启用 vsCode 相关插件

路径：文件 -> 首选项 -> 拓展

  • 代码格式化： Prettier
  • vue 代码块高亮：Vetur

加入配置代码

路径：文件 -> 首选项 -> 设置

（新版 vsCode 路径：文件 -> 首选项 -> 设置 -> 搜索 Associations）

在 settings.json -> 工作区设置 加入以下配置代码 

{
  "prettier.semi": false,
  "prettier.singleQuote": true,
  "eslint.options": {
    "configFile": ".eslintrc.js"
  },
  "eslint.validate": ["javascript", "html", "vue"],
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  // vue 格式化
  "vetur.format.defaultFormatter.html": "js-beautify-html",
  "vetur.format.defaultFormatterOptions": {
    "js-beautify-html": {
      "wrap_attributes": "force"
    },
    "prettier": {
      "singleQuote": true, // 使用单引号
      "semi": false // 是否在某一行末加分号
    }
  },
  "path-autocomplete.pathMappings": {
    "@": "${folder}/src"
  },
  "git.ignoreLimitWarning": true
}

实景图：

完成以上配置后重启 vsCode，运行项目，执行 Ctrl+S（保存快捷键）页面代码则会自动格式化

如涉及文件过多没有格式化，可通过 执行npm run lintFix 完成批量格式化

Tips: 插件在安装后需 "应用" 该插件，功能无法使用或异常 可重启 vsCode 解决
